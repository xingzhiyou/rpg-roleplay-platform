# 参数调节（模型设置）

## 这页能干嘛

「模型设置」页调整发送给 AI 模型的采样参数，控制 GM 输出的随机程度、长度、重复抑制等。这些参数影响所有 API 调用，是调整 GM「说话方式」的旋钮。

入口：顶栏「设置」→「模型参数」（路由：`#settings-modelparams`）。

> 这是旧版路由 `settings-params` 的重定向入口，实际页面路由为 `settings-modelparams`。

详细说明请见[模型参数调优](./settings-modelparams.md)。

---

## 快速参考

### 预设一键切换

| 预设 | 适用场景 |
|---|---|
| 平衡 | 通用默认，兼顾流畅和创意（temperature 0.78） |
| 保守 | 规则裁定、系统解析（temperature 0.40） |
| 创意 | 开放世界、诗意风格（temperature 1.00） |
| 确定 | 调试和复现（temperature 0.10） |
| 自定义 | 手动调整后自动切换 |

### 主要参数

| 参数 | 默认值 | 说明 |
|---|---|---|
| Temperature | 0.78 | 随机性：越高越天马行空，0 最确定 |
| Top-p | 0.92 | 累积概率截断，0.90–0.95 典型值 |
| Top-k | 40 | 从 Top K 个词采样，0 = 关闭 |
| Repetition Penalty | 1.15 | 抑制重复用词，1.0 = 无效果 |
| 最大输出 Tokens | 1024 | 单轮最大输出长度 |
| 上下文窗口 | 16K | 每次请求携带的历史上限 |
| NSFW | 含蓄 | 成人内容控制（禁止/含蓄/开放/露骨） |

页面底部「**预览 JSON**」实时显示实际发送给 API 的参数格式。
