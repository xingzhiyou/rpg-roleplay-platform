# 世界书

## 这页能干嘛

「世界书」是 GM 的背景知识库——记录剧本世界中的地名、人物关系、势力、规则约束等设定。导入剧本后，系统自动从章节内容提取世界书条目；你也可以在此手动添加或编辑。

世界书条目在每轮 GM 回复前，会按当前场景的相关性选择性注入上下文，让 GM「知道」这个世界的规矩和地理。

入口：剧本管理 → 选中剧本 → 详情面板「**世界观**」标签。

## 关键概念

### 条目结构

每条世界书条目包含：

- **标签（label）**：条目名称，GM 在匹配场景时根据此标签触发
- **世界线键（wl_key）**：所属世界线
- **是否主要（★）**：主要条目优先级更高，更容易被注入
- **内容**：具体设定文字

### 触发机制

GM 不是每轮都注入所有条目，而是根据当前对话内容的语义相关性动态选择。设置了关键词或提高了重要度的条目更容易被触发。

## 常见任务

### 查看世界书条目

剧本管理 → 选中剧本 → 详情面板「**世界观**」标签，列出该剧本所有世界书条目。

### 游戏中查看实时世界书状态

进入游戏 → 右侧面板「**世界书**」标签（在游戏中实时显示当前场景的动态状态，如位置、时刻、天气、世界规则）。

---

## 常见问题

**世界书条目太多 GM 会不会乱？**
不会。系统按相关性只选取当前场景需要的条目注入，不会全量注入。可以在「记忆设置」调整注入 token 上限。

**怎么确保某个重要设定 GM 每轮都看到？**
在游戏记忆面板手动添加「固定记忆」，固定记忆每轮必然注入，优先级高于世界书。
