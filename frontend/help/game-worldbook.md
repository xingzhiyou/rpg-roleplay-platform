# 世界书面板

## 这页能干嘛

世界书面板让你在游戏进行中随时查看和调整当前场景的世界设定参数——包括玩家所在位置、当前时刻、天气，以及这个剧本的世界规则约束。面板里显示的内容会直接影响 GM 的叙事背景，你也可以点击字段直接修改，让 GM 的下一轮回复反映最新的场景状态。

![世界书面板](./screenshots/game-worldbook-panel.png)

---

## 关键概念

### 世界书是什么

世界书是剧本的背景设定库。导入剧本时，系统会自动从剧本内容里提取地名、时代背景、关键规则等信息，生成世界书条目。GM 在每一轮叙事前，会从世界书里召回与当前场景相关的条目作为参考——这就是 GM 为什么"知道"这个世界的规矩和地理的原因。

### 地点·时刻（可编辑字段）

面板顶部显示四个实时字段：

| 字段 | 含义 |
|------|------|
| 所在 | 玩家当前位置（如"柏林·夏洛滕堡"） |
| 时刻 | 故事内时间（如"1935 春·清晨"） |
| 天气 | 当前天气（如"细雨"） |
| 阶段 | 当前所处的剧情阶段（如"柏林暗流篇"） |

这四个字段都支持**点击直接编辑**：单击字段值，输入新内容，回车或点击外部区域即可保存，GM 下一轮会读到更新后的值。

### 世界规则（约束列表）

剧本预设的世界逻辑约束，由剧本作者或自动提取写入，玩家无法在游戏中修改。GM 会将这些规则作为硬限制参考——比如"魔法在特定区域无效"或"某角色在故事里不可被击杀"。如果某个剧本没有配置世界规则，这里会显示"暂未配置"。

### 本轮重要词条

当前回合里 GM 已知的重要事件词条，以标签形式展示。这些词条来自 `state.world.known_events`，由 GM 每轮叙事后自动更新，反映本轮对话中出现的关键词/事件。

## 常见任务

**修改当前位置**
点击"所在"行的值 → 输入新位置 → 按回车。GM 下一轮叙事时会以新位置为准。

![世界书字段内联编辑](./screenshots/game-worldbook-field-edit.png)

**修改故事内时间**
点击"时刻"行的值 → 输入新时间描述（建议格式：年代+季节+时段，如"1942 冬·深夜"）→ 回车保存。

**查看世界规则**
直接看"世界规则"区域的约束列表。如果列表为空，说明当前剧本没有预设硬约束，GM 的叙事自由度更高。

**查看本轮关键词条**
看面板底部的"本轮重要词条"标签区，这里实时反映本轮 GM 已知的重要事件。

## 常见问题

**世界书里的设定和我在对话里告诉 GM 的设定，哪个优先？**
通常对话里的最新指令优先，但世界书里的"世界规则"是硬约束，GM 不会违背它们（比如剧本规定某角色不死，即使你在对话里说"杀死他"，GM 也不会写出对应结果）。

**游戏内改了位置/时刻，会影响存档吗？**
会。修改后系统会自动保存到当前存档的运行时状态，再次加载存档时位置和时刻保持修改后的值。

**世界书条目和剧本导入时自动生成的有什么区别？**
本质上是同一套数据。剧本导入时，系统提取器会自动生成世界书条目（地名、人名、关键规则等），这就是游戏里世界书的数据来源。游戏中这个面板只展示当前场景相关的动态字段（位置/时刻/天气），完整的世界书条目库在平台的剧本管理页里可以查看和编辑。

**修改天气/时刻后 GM 没有反映，怎么办？**
发送一条消息触发新一轮 GM 回复即可。修改在存档里已经生效，但要等下一轮 GM 才会读到并在叙事里体现。
