# 角色管理（剧本 NPC 卡）

## 这页能干嘛

「角色管理」是剧本级 NPC 角色卡的查看和编辑入口。这里的 NPC 卡是从剧本文本中由 LLM 提取，或由你手动创建的，绑定到某本具体剧本。

入口：剧本管理 → 选中剧本 → 详情面板「**NPC 角色卡**」标签。

![NPC 角色卡列表](./screenshots/characters-npc-list.png)

---

## NPC 角色卡与用户角色卡的区别

| | NPC 角色卡 | 用户角色卡 |
|---|---|---|
| 代表谁 | 剧本里的配角 | 你扮演的 PC |
| 归属 | 绑定某本剧本 | 账户级，跨剧本共享 |
| 入口 | 剧本管理 → 详情 → NPC 角色卡 | 顶栏「角色卡」页 |
| 可见范围 | 剧本内 / 私有 / 公开 | 私有 / 公开 |

---

## 角色卡字段

与用户角色卡字段相同，但 NPC 卡额外有「**首次揭示章节**」字段（章节闸——在该章之前 GM 不会主动引用此 NPC）。详见[角色卡](cards.md)的字段说明。

---

## 常见任务

### 查看 NPC 列表

剧本管理 → 选中剧本 → 详情面板「**NPC 角色卡**」标签，列出所有提取的 NPC。

### 新建 NPC 卡

在 NPC 标签页右上角「**新增 NPC 卡**」→ 填写字段 → 创建。

### 编辑 NPC 卡

点击某条 NPC → 展开详情 → 修改字段 → 保存。

### 删除 NPC 卡

NPC 卡的删除入口在本页（剧本管理的 NPC 标签），不在「角色卡」主页。

---

## 常见问题

**NPC 卡列表是空的？**
说明该剧本还没有运行 KB 提取，或提取未完成。先建立向量索引，再在详情面板运行知识提取。

**在「角色卡」主页看不到 NPC？**
NPC 卡归属于剧本，不在用户角色卡列表里。入口在剧本管理 → NPC 角色卡标签。
