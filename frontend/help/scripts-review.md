# 剧本复核

## 这页能干嘛

「剧本复核」页用于检查从剧本中提取的知识库（KB）质量。你可以在这里查看 LLM 从章节文本中提取的实体（角色、地点、事件等）摘要是否准确，并对错误的条目进行修正或删除。同时也可以预览该剧本的世界书条目。

入口：剧本管理 → 选中剧本 → 标题下方「**剧本复核**」链接（或详情面板操作菜单）。

![剧本复核界面](./screenshots/scripts-review-main.png)

---

## 页面内容

### 实体表格

| 列 | 说明 |
|---|---|
| 实体键（logical_key） | 实体的唯一标识符 |
| 摘要 | LLM 生成的描述，可手动修改 |
| 操作 | 改摘要 / 删 |

### 世界书条目列表

页面下方展示该剧本的世界书条目，显示标签名（★ 表示主要条目）和世界线键。

---

## 常见任务

### 修正错误摘要

1. 找到摘要不准确的实体行
2. 点「**改摘要**」，编辑框出现
3. 修改内容后点「**存**」

### 删除误识别实体

找到错误提取的实体（如广告文字或章节标题被误识别为角色名）→ 点「**删**」→ 确认删除弹窗。

> 删除不可撤销。误删后需重新运行 KB 提取恢复（手动修改会丢失）。

---

## 常见问题

**摘要是空白的？**
该实体在章节里出现太少，LLM 未能生成有效摘要。可手动点「改摘要」补充。

**复核完后怎么让 GM 使用修正后的摘要？**
保存修改即时生效，下次开始游戏时 GM 会使用修正后的摘要。

**章节切分有问题，也在这里改吗？**
不是，章节层面的操作（重命名、拆分、合并、整本重切）在「剧本管理 → 查看章节」弹窗里操作。
