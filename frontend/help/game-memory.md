# 记忆面板

## 这页能干嘛

记忆面板是你在当前游戏里的"大脑外挂"。它把 GM 积累的背景知识、你自己记下的线索、以及本轮从原著/历史里检索到的参考资料，集中在一个地方给你看。你也可以在这里手动钉住关键事实，确保 GM 每轮都能记住它们。

![游戏记忆面板](./screenshots/game-memory-panel.png)

---

## 关键概念

### 当前目标
面板顶部显示**主线任务**和**当前小目标**。主线任务来自剧本设定，当前小目标由 GM 在推进剧情时动态更新。

### 固定记忆（pinned）
标记了图钉图标的条目。每一轮 GM 生成回复时，这些条目**必然会被注入上下文**，不会因为太久远而被遗忘。适合放：绝对不能忘的角色设定、你和某 NPC 的约定、关键道具线索。

点击右上角的 **+** 按钮可以新增固定记忆；点击条目右侧的 **×** 可以解除钉住（解除后该条内容不再保底注入，但不会被删除）。

### 事实库（facts）
由 GM 自动提取并写入，无法手动编辑。每次 GM 完成一轮叙事，系统会从对话里抽取关键事实（NPC 状态变化、发生了什么事）存入这里，作为后续轮次的参考库。

### 玩家笔记（notes）
供你自由记录的空间，GM **不保证**每轮都会读到（不像固定记忆那样强制注入）。适合放思路猜测、想法便签、对话摘录。点 **+** 添加，点条目右侧 **×** 删除。

### 召回（本轮检索到的参考资料）
面板底部的灰色区域。每一轮 GM 回复后，系统会把这一轮从知识库/历史对话里找回来的文本片段展示在这里，让你了解 GM 当前参考了哪些背景信息。右上角的数字是本轮召回的段落数。

---

## 常见任务

**钉住一条重要信息**
点击"固定记忆"区域右上角的 **+** → 在弹窗里输入文字 → 回车确认。该条目立即出现在固定记忆列表，下一轮 GM 就会读到它。

![添加固定记忆](./screenshots/game-memory-add-pinned.png)

**解除钉住**
找到想解除的条目 → 点右侧的 **×** → 确认弹窗。解除后该条目从固定记忆移除，不再保底注入。

**添加玩家笔记**
点击"玩家笔记"区域右上角的 **+** → 输入内容 → 确认。笔记不会影响 GM 的必读列表，只是你的私人备忘。

**查看本轮 GM 参考了什么**
直接看面板底部的"召回"区域，里面是原文或历史对话的片段。

## 常见问题

**固定记忆和玩家笔记有什么区别？**
固定记忆每轮必进 GM 上下文（有字数配额，请勿堆砌）；玩家笔记只供你参考，GM 不一定每轮都读。

**固定记忆和"平台设置 → 记忆配置"有什么区别？**
平台设置里的记忆配置是**全局默认值**，决定新存档的初始行为（比如短期记忆保留多少轮）。游戏内记忆面板操作的是**当前这局游戏的运行时状态**，修改只影响本存档，不会改变平台默认配置。

**GM 什么时候会自动写入事实库？**
每一轮 GM 完成叙事后，系统后台会自动提取对话中出现的关键事实并写入事实库。你不需要做任何操作，等下一轮刷新面板即可看到新增内容。

**固定记忆能放多少条？**
没有硬性上限，但每轮注入有 token 预算。固定记忆过多时，优先级低的条目可能在高压轮次里被压缩。建议把真正关键的设定放这里，杂项放玩家笔记。
