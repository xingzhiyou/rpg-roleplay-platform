# 输入面板（游戏输入框）

## 这页能干嘛

游戏主界面底部的输入区是你与 GM 对话的核心入口，支持普通对话、斜杠命令、角色 @mention 和文件附件。

入口：进入游戏 → 界面底部输入框区域。

![游戏输入框](./screenshots/input-composer.png)

---

## 普通对话

在输入框里写你的行动、对话、描述，按「**回车**」发送。GM 会流式输出回应，可随时点「停止」中断。

- **回车**：发送
- **Shift + 回车**：输入框内换行
- **「继续」按钮**：无需输入，让 GM 直接推进一段剧情（玩家被动时使用）

---

## / 命令

输入框里输入 `/` 会弹出命令菜单，用方向键选择、回车确认、Esc 取消。

| 命令 | 说明 |
|---|---|
| `/set 描述` | 用自然语言批量强制写入游戏参数（如 `/set 时间=次日清晨；地点=码头`） |
| `/loc 地点` | 快速更新所在位置 |
| `/time 时刻` | 推进故事内时间 |
| `/rel 角色 关系` | 更新与某角色的关系状态 |
| `/var 变量=值` | 设置世界线自定义变量 |
| `/pin 文本` | 添加固定记忆，GM 每轮都会参考 |
| `/note 文本` | 玩家私人笔记，不影响 GM |
| `/memory normal/deep/off` | 切换记忆检索模式 |
| `/permission default/review/full_access` | 切换 GM 写入权限 |
| `/save` | 手动保存当前存档 |
| `/retry` | 重试 GM 上一轮输出 |
| `/status` | 显示当前状态摘要 |
| `/debug` | 查看上一轮 GM 的内部检索结果 |

---

## @ 提及角色

输入框里输入 `@` 弹出角色列表，选择后插入 `@角色名`，GM 会将其视为针对该角色的发言或行动。

---

## 附件（+ 按钮）

点击输入框左侧「**+**」打开附件菜单：

**本地**
- **文件 / 文本**（TXT · MD · JSON）：上传文档，GM 本轮参考内容
- **图片**（PNG · JPG · WEBP）：上传图片，GM 多模态理解

**剧本资源**
- **插入章节**：从剧本目录搜索一章节注入上下文
- **角色卡**：搜索或拖入角色卡，让 GM 参考详细设定
- **世界书条目**：全文搜索世界书，注入匹配条目

**能力**
- **MCP 工具**：挂载已配置的 MCP 服务器工具
- **Skill 包**：挂载规则裁定包（如骰子规则、5E 规则集）
- **计划模式**：预演叙事路径，不写回存档状态

---

## 权限控制

工具栏显示当前 GM 写入权限，点击切换：

| 模式 | 说明 |
|---|---|
| **只读 · 纯叙事** | 所有状态写入需玩家手动审批 |
| **默认权限** | 位置/时间等白名单字段自动写入，其他需审批 |
| **自动审查** | 白名单 + 关系/世界变量自动写入，其他需审批 |
| **完全访问** | 除硬黑名单外全部自动写入 |

GM 提出写入申请时，输入框上方会出现待确认条目，可逐条允许或拒绝。

---

## 模型切换

工具栏右侧显示当前 GM 使用的 AI 模型，点击可切换。颜色圆点：绿=可达，橙=降级，红=不可达。

---

## 上下文用量

工具栏右侧有圆环进度指示，显示当前上下文占用比例。点击查看详细分布（记忆 / 世界书 / 对话历史等各部分 Token 占用）。超过 70% 变橙色，超过 90% 变红色，建议存档后开启新对话。
