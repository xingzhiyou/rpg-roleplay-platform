# 时间线面板

## 这页能干嘛

时间线面板（标签栏上显示为"世界线"）让你看清当前剧情走到了哪里：剧本原著预设了哪些关键节点、你实际走过了哪些阶段、哪些情节已经发生或偏离了原著轨道。面板还会显示"世界线收束·锚点"数据，让你了解 GM 对世界线的管控压力，预判接下来哪些情节是"不可绕过"的。

![时间线面板](./screenshots/game-timeline-panel.png)

## 关键概念

### 世界树是什么

世界树（Worldtree）是这个游戏的核心底层结构。每一次游戏状态的推进都会生成一个提交节点（commit），这些节点像 git 的提交历史一样形成一棵有向无环图——这就是世界树的"脊柱"。

从玩家角度理解：世界树分两个维度：
- **规范世界线**（主线）：原著剧本里"应该发生"的故事走向，由剧本作者或系统提取，固定只读。
- **玩家实际足迹**：你在游玩过程中实际走过的路径，会随着你的选择偏离或贴近规范线。

### 剧本期望线（上方轨道）

以时间轴形式展示剧本的关键锚点，每个节点包括阶段名和对应的故事时间/章节范围。节点状态有三种：

- **已度过**（绿色实心点）：你已经经历了这个剧情阶段
- **当前**（高亮蓝色，带"当前"标签）：你现在所在的剧情位置
- **待解锁**（灰色，带"待解锁"标签）：还没有走到的剧情阶段

如果当前剧本没有预设时间线（如纯自由创作存档），这里会显示"此剧本没有预设时间线"。

### 实际足迹线（下方轨道）

记录你在这个存档里实际走过的每个阶段（Phase），每个阶段条目包含：

- 阶段编号（Phase N）
- 阶段标签/名称
- 对应的 turn 范围（第几轮到第几轮）
- 故事时间标签（如果有）
- 阶段摘要（如果 GM 写入了）

点击一个有关键事件的阶段，可以展开查看该阶段里发生的重要事件列表（带 turn 编号）。

### 世界线收束·锚点

面板底部的第三个区域，是这个存档的"命运压力表"。

**什么是锚点？** 锚点是剧本里那些"不论玩家怎么操作，GM 都会想办法让它发生"的关键情节节点。比如主角一定会在某处遇到某人，某场事件一定会在故事里发生。

**锚点状态分四种：**

| 状态 | 含义 |
|------|------|
| 待发生 | 锚点还没触发，GM 在等待合适时机 |
| 已发生 | 情节按原著正常发生 |
| 变体 | 情节发生了，但走向和原著不同 |
| 绕过 | 情节被完全跳过，可能因为玩家走了完全不同的路径 |

**偏离度（drift）**：0% 表示你完全按原著走，100% 表示高度偏离。GM 会根据偏离度决定是否施加收束压力，把故事往关键锚点方向引导。

**必发生锚点**：如果某个锚点标记为"必发生"（fatal），无论偏离度多高，GM 都不会放弃把它引入故事。

## 常见任务

**查看当前剧情走到哪里了**
看"剧本期望线"轨道，找到带"当前"标签的高亮节点；再看"实际足迹线"轨道最底部（最新）的阶段条目。

![时间线期望线节点](./screenshots/game-timeline-expected.png)

**查看已经走过的历史**
在"实际足迹线"里，点击某个历史阶段条目，可以展开查看该阶段的关键事件列表（如果 GM 记录了的话）。

**了解接下来有哪些"强制剧情"**
看"世界线收束·锚点"区域的"待发生锚点"列表，尤其注意标有红色"必发生"标签的锚点——这些是 GM 一定会安排进剧情的关键节点。

**了解偏离度压力**
看锚点区域的"平均偏离度"进度条。绿色（0-30%）表示你基本贴近原著；橙色（30-60%）表示有明显偏离；红色（60%+）表示 GM 会开始主动收束，把剧情拉回轨道。

**按剧情阶段查看锚点分布**
看"按阶段分布"轨道，每个阶段显示已收束/总数和收束压力百分比，颜色越红说明该阶段压力越大。

## 常见问题

**世界线和存档分支有什么关系？**
每个存档对应一条玩家世界线。存档分支功能（平台「存档」页）允许你从某个时刻分叉出一条新的游戏线，相当于在世界树上创建一个新分支。不同分支之间共享剧本的规范期望线，但各自有独立的实际足迹和锚点状态。

**实际足迹里没有内容，只显示"暂无足迹"，正常吗？**
正常，这说明当前存档还没有完整走完一个阶段（Phase）。阶段数据是后端在游戏推进过程中按轮次积累的，通常在发送几轮消息后才会出现第一个 Phase 记录。

**我的剧情明显偏离了，偏离度却还是绿色，为什么？**
偏离度是基于锚点的触发情况计算的平均值，不是实时分析你的对话内容。如果所有锚点都还是"待发生"（没有明确触发过或绕过），偏离度可能暂时较低。随着剧情推进会逐渐准确。

**时间线加载失败怎么办？**
面板会显示错误提示和排查建议（后端是否在运行/账号是否有权限）。通常刷新页面或等后端恢复后重新进入游戏界面即可。如果问题持续，检查平台首页的服务状态。

**"按阶段分布"里的"收束压力"是什么意思？**
收束压力由该阶段的剩余待发生锚点比例和重要度计算得出。压力高意味着 GM 在这个阶段会更主动地引导故事走向锚点方向，你的"自由行动"空间会相对变小。
