# 系统管理

## 这页能干嘛

「系统管理」是平台管理员专用的后台，提供对全平台用户、用量、安全、运维的管理能力。**只有角色为「管理员」的账号才能访问**，普通用户访问时会看到权限拒绝页面。

入口：顶栏「全部功能」→「系统管理」，或直接导航 `#admin-users` 等子页。

## 用户管理

查看所有用户的用户名、显示名、角色（管理员/普通用户）、状态、最后登录时间、30天Token消耗、活跃Session数。支持按名称搜索、按角色和状态过滤。

**可执行操作：**

| 操作 | 说明 |
|---|---|
| **停用** | 用户无法登录，数据保留，可随时恢复 |
| **恢复** | 重新允许已停用用户登录 |
| **强制下线** | 使该用户所有 Session 立即失效 |
| **升为管理员** | 授予系统管理权限 |
| **降为普通用户** | 撤销管理权限 |

所有危险操作都有确认对话框。

## 全局用量

时间范围可选（7 / 14 / 30 / 90 天）：
- **总览卡片**：总请求数、总 Token（入+出）、总成本（USD）
- **按用户排行**：谁用得最多，含 Token 量、成本和占比百分比
- **按 API 分布**：各供应商各用多少
- **按天趋势**：日粒度 Token 柱状图

---

## 审计日志

记录所有管理员操作（`user.deactivate` / `config.save` / `invite.create` 等），可按操作类别过滤。字段包含：时间、操作者、操作类型、目标、详情、IP。

---

## 系统健康

每 30 秒自动刷新，显示：
- **数据库**：连接状态、查询延迟（ms）
- **内存**：进程 RSS 内存占用
- **磁盘**：已用空间百分比（超 90% 变警告色）
- **进程**：PID 和已运行时长

---

## 系统日志

按行数过滤（50 / 100 / 200 / 500 行）和按级别过滤（全部 / ERROR / WARN / INFO）。ERROR 行红色高亮，WARN 行橙色高亮。可下载为 `.log` 文件。

---

## 注册与邀请

**注册模式**（三选一）：
- **开放注册**：任何人可直接注册
- **仅邀请**：只有持有效邀请码才能注册
- **关闭注册**：完全关闭新用户注册

还可以分别开关「邮箱验证」和「自动审批」。

**邀请码生成：** 点「生成邀请码」→ 填写数量（1-10）、过期天数、备注（可选）→「生成」。列表中显示每个码的状态（可用 / 已使用 @用户名 / 已过期），未使用的码可随时删除。

---

## 安全配置

- **速率限制**：每IP最大请求数、每用户最大请求数、时间窗口（分钟）。修改需重启生效。
- **密码策略**：最小长度、是否要求数字
- **Session 策略**：Session 超时天数
- **登录锁定策略**：连续失败次数阈值、锁定时长（分钟）
- **IP 黑名单**：每行一个 IP 或 CIDR 段（如 `192.168.1.0/24`）

---

## 维护模式 + 服务重启

**维护模式**：开启后所有用户看到维护公告，可在此填写公告内容和维护开始时间。

**服务重启**：向后端发送优雅重载信号，重启约 5-15 秒。建议先开维护模式再重启。

---

## 合规功能（DMCA / CSAM / AUP）

- **DMCA 下架队列**：录入通知、执行下架/恢复/拒绝操作，支持录入反通知（自动设 10 天恢复计时）
- **Strike 记录**：为用户添加 DMCA Strike（3 次触发账户终止流程）
- **CSAM 举报**：记录举报、标记决定（成立/升级/不成立），成立需按规程向 NCMEC CyberTipline 上报
- **AUP 用户行动**：搜索用户执行暂停（临时）、解封、永久终止（不可逆）
- **反馈审查**：查看用户提交的反馈，标记「OK / 标垃圾 / 终止账号(NSFW)」

## 部署配置（仅管理员可见）
