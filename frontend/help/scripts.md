# 剧本管理

> **注意**：列表中标有"[内部]"或"敬请期待"标签的剧本是**开发中的功能预告**，目前无法开始游戏。点击后会显示功能说明，正式上线前不可用。

## 这页能干嘛

「剧本管理」是你导入和管理小说剧本的地方。上传一部 TXT 文件后，系统会自动识别章节结构并切分，之后建立向量索引，就可以基于该剧本开始新游戏。

![剧本管理主界面](./screenshots/scripts-list.png)

---

## 关键概念

### 三种视图

- **我的剧本**（默认）：列出你已导入的全部剧本，可查看章节数、字数、切分质量、关联存档数。
- **导入**：上传 TXT 文件的全屏页面，包含章节切分预览与规则选择。
- **在线剧本库**：浏览其他用户公开分享的剧本（入口在上方导航）。

### 章节切分规则（7 种）

导入时系统自动嗅探最合适的规则（置信度 ≥ 85% 显示绿色）。可手动指定：

| 规则 | 适用场景 |
|---|---|
| 自动识别 | 优先推荐，覆盖大多数中文网文格式 |
| 语料章节 | 带 `第X章` 原始语料格式 |
| 中文章节 | 「第X章」「第X回」等中文编号 |
| 英文章节 | Chapter X / CHAPTER X |
| 数字点号 | `1.`、`2.` 开头 |
| 括号编号 | `（1）`、`【1】`等 |
| 自定义 | 自定义正则，手动填写匹配模式 |

### 向量索引

切分完成后还需要「建立向量索引」，GM 才能检索到相关段落用于游戏。点击剧本详情面板右上角「建立向量索引」即可启动后台任务。

## 常见任务

### 导入一本新剧本（TXT 上传）

1. 点击右上角「**导入剧本**」，进入导入页面。
2. 上传 `.txt` 文件（UTF-8 编码，建议去掉广告行和版权声明）。
3. 系统显示「章节切分预览」弹窗，展示章节数、总字数、置信度，以及前几章的标题列表。
4. 如果置信度偏低或章节错位，点「**换规则重试**」切换规则，直到预览正常。
5. 确认后点「**确认导入**」，系统在后台完成全量切分。

![章节切分预览弹窗](./screenshots/scripts-import-preview.png)

### 导入剧本包（ZIP 格式）

点击右上角「**导入剧本包**」，接受 `.zip` 打包文件（含章节、角色卡、世界书），适合一键还原从他人处拿到的导出包。

### 查看剧本详情

点击列表中某条剧本 → 下方展开详情面板，包含以下标签：

- **概览**：标题、章节数、字数、切分规则、置信度、关联存档数
- **参数**：脚本级覆盖设定（`script_overrides`），如 GM 行为调整
- **世界观**：世界书条目预览
- **NPC 角色卡**：该剧本提取的 NPC 列表
- **时间线**：故事阶段锚点

### 建立向量索引

选中剧本 → 详情面板右上角「**建立向量索引**」→ 等待进度完成。索引建立后 GM 才能通过 RAG 检索相关段落。

### 公开 / 私有切换

选中剧本 → 详情面板操作菜单 →「**公开分享到剧本库**」或「取消公开分享」。公开后其他用户可以浏览并导入你的剧本。

### 查看章节列表

详情面板右上角「**查看章节**」（或行操作下拉「查看章节 / 重切分」）→ 打开章节管理弹窗，左侧章节列表，右侧展示章节正文。

### 导出剧本包

详情面板操作菜单 →「**导出剧本包**」→ 下载 `.zip` 文件，包含章节、角色卡、世界书。

### 删除剧本

详情面板操作菜单 →「**删除**」→ 确认弹窗。删除会同时清理对应的向量索引和存档，不可撤销。

## 常见问题

**导入后章节数不对，怎么办？**
打开「查看章节」→ 选择「整本重切」，换一个规则试试。推荐先试「中文章节」，再试「自定义」并填写正则。

**置信度低于 70% 是什么意思？**
系统识别章节标题的把握很小，实际切分可能错位。建议在预览里逐条检查，手动选规则修正。

**为什么建了索引还是看不到 NPC 角色卡？**
向量索引建立完成后，还需要在详情面板「NPC 角色卡」标签手动触发 KB 提取。提取结束后才会生成 NPC 角色卡列表。

**删除剧本后存档还在吗？**
不在。删除剧本会同时删除关联的所有存档及向量索引，操作前请确认。
